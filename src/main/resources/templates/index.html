OIDC Session Management Demo

About This Demo

This application demonstrates OIDC (OpenID Connect) session management features using Keycloak as the OpenID Provider.

Features Demonstrated:

  • Session Monitoring: Real-time monitoring of OIDC session status using iframe-based session checks
  • Session State Tracking: Display and track the session_state parameter
  • Automatic Session Detection: Detect when user logs out from another tab/application
  • RP-Initiated Logout: Properly logout from both application and Keycloak
